# Qu::Utils

Utils: useful class or methods used in qu-* projects.

## Installation

Add this line to your application's Gemfile:

    gem 'qu-utils'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install qu-utils

## Usage

```
require 'qu/utils'

Qu::Utils::long_seq_wrap(long_seq)
```
